# ☕ Coffee Sales Dashboard (Excel)
<h3> Project Overview </h3>

This project is an interactive Excel dashboard built to analyze coffee sales performance. The goal was to transform raw transactional data into clear, actionable insights using Excel’s analytical and visualization capabilities.
The workflow covers data cleaning, feature engineering, analysis, and storytelling through dashboards.

<h5> Objectives </h5>

-Clean and standardize raw coffee sales data 

-Create derived columns to enrich analysis

-Analyze sales trends and performance using Pivot Tables

-Build an interactive dashboard to communicate insights clearly

<h5> Tools & Techniques Used </h5>

Microsoft Excel,
Data Cleaning & Preparation,
Derived Columns,
XLOOKUP,
INDEX / MATCH,
Pivot Tables & Pivot Charts,
Slicers for interactivity,
Data Visualization (bar chart, line chart)

<h5> Data Preparation </h5>

Removed duplicates and handled missing values,
Standardized date formats and categorical fields,
Created derived columns such as:
-Total Sales
-Customer and Product attributes

<h5> Analysis & Features </h5>

Pivot Tables used to analyze:

Total Sales over Time

Sales by country

Top 5 customers

*Lookup functions (XLOOKUP, INDEX) used to:*

Merge related datasets

Retrieve product and customer details dynamically

*</h5> Dashboard includes: </h5>*

Interactive charts

Slicers to filter by order date, coffee type, coffee size and loyalty card

*<h5> Key Insights </h5>*

Identified top-performing coffee beans over time

Highlighted coffee sales in countries and customers

Enabled quick comparison across time periods and categories

<h5> Dashboard Preview </h5>

The final Excel dashboard allows users to:

Filter sales data dynamically

Explore trends at a glance

Understand performance without digging into raw data

How to Use

Download the Excel file from the repository

Open in Microsoft Excel (desktop version recommended) or WPS Excel.

Use slicers to interact with the dashboard

Explore insights through charts and pivot tables
